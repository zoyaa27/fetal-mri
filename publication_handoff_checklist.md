# Publication Handoff Checklist

This checklist turns the publication-readiness audit into a handoff packet for
the radiologist collaborators. It is intentionally operational: every row maps a
reviewer expectation to a manuscript section, Required evidence, and Owner.

## Results Endpoint Checklist

| Endpoint                       | Manuscript section                | Required evidence                                                                                                                                                                                                                                                                                                                                                                                            | Owner                  | Go / no-go                                                 |
| ------------------------------ | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | ---------------------- | ---------------------------------------------------------- |
| Per-parameter agreement        | FeTA and institutional validation | MAE / MAPE for z-scored parameters, Bland-Altman plots, ICC where repeated readers exist, explicit exclusion of third-ventricle z-score reporting, and FeTA 2024 source metadata (PMID 41564637, DOI 10.1016/j.media.2026.103941).                                                                                                                                                                           | Analyst                | Go only after FeTA and local measurements are locked       |
| External robustness            | FeTA subgroup analysis            | Stratify by site, vendor, field strength, super-resolution strategy, acquisition site, and image quality tier.                                                                                                                                                                                                                                                                                               | Analyst                | Go only if subgroup failures are disclosed or mitigated    |
| Cohort flow / missingness      | Methods and Results               | `summarizeValidationCohortFlow` cohort flow and missing-data summary with total cases, included cases, excluded cases, exclusion reasons, reference-standard availability, prediction availability, pathology-label availability, complete-case denominators, and per-parameter missingness.                                                                                                                 | Analyst                | Go only after exclusion reasons are locked                 |
| Pathology contrast             | FeTA subgroup analysis            | Welch two-sample comparison of calculator z-score distributions between pathological and neurotypical FeTA subsets, with mean difference, confidence interval, degrees of freedom, and significance-by-CI flag.                                                                                                                                                                                              | Analyst                | Go only after pathology labels are locked                  |
| Discrimination and calibration | Primary results                   | ROC-AUC confidence interval, PR-AUC, sensitivity / specificity at locked thresholds, Wilson confidence intervals for proportions, calibration-in-the-large, calibration slope, and Brier score.                                                                                                                                                                                                              | Analyst                | Go only after thresholds are frozen before analysis        |
| Sample-size / precision plan   | Methods - statistical analysis    | `estimateDiagnosticAccuracyPrecisionSampleSize` for sensitivity / specificity positive-case and negative-case requirements by Wilson interval half-width, plus `estimatePairedMeanDifferenceSampleSize` for paired reader-study timing and completeness endpoints.                                                                                                                                           | Analyst + PI           | Go only after target half-widths and expected effects lock |
| Clinical utility               | Secondary results                 | decision-curve net benefit across plausible follow-up thresholds and a plain-language threshold rationale.                                                                                                                                                                                                                                                                                                   | Analyst + PI           | Go only if decision threshold matches clinical workflow    |
| Pre/post report audit          | QI / implementation results       | Average time to report, all-required-measurement completion, explicit z-score documentation, explicit percentile documentation, and recommendation congruence.                                                                                                                                                                                                                                               | Analyst + PI           | Go only after 100 baseline and 100 post-tool reports       |
| Reader study                   | QI / implementation results       | paired within-reader / within-case deltas and paired delta confidence intervals for reader-study timing, report completeness, z-score documentation rate, and recommendation congruence; Cohen's kappa for two-reader categorical labels, Fleiss's kappa for three-plus-reader categorical labels, ICC(2,1) for repeated measurements, raw NASA Task Load Index scoring, and System Usability Scale scoring. | PI + radiologists      | Go only after IRB / QI determination                       |
| Source-data final lock         | Supplement / appendix             | Dovjak/Woitek/extra-axial CSF implementation byte-check countersignature, third-ventricle raw-threshold policy, and Chiari calibration status.                                                                                                                                                                                                                                                               | Clinician collaborator | Go only after unresolved source checks are marked reviewed |

## Go / No-Go Summary

The project is ready for radiologist review when the application gates are green
and this checklist is present. The project is ready for manuscript submission
only after the source-data final lock, validation_analysis_lock.md, IRB / QI
determination, FeTA or equivalent external validation, cohort flow and
missing-data summary, institutional reader study, sample-size / precision plan,
calibration table, decision-curve net benefit table, pre/post report-audit
metrics, and reviewer-standard checklist rows are filled with actual data.
